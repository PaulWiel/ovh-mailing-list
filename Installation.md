# Installation #

  * Télécharger et décompresser l'archive ZIP
  * Copier le contenu de l'archive sur le serveur FTP de l'hébergement OVH
  * Le répertoire peut être renommé à souhait (à condition de spécifier l'url dans la configuration)
  * Paramétrer le CHMOD du fichier config.php à 0444
  * Paramétrer le CHMOD des fichiers temp\_sub.php et temp\_unsub.php à 0666
